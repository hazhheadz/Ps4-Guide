    Required Reading

    This is for installing the dumps of discs you have made for convienence, please do not use this guide for piracy!

    Usually games, patches and DLC are distributed together. You should install them in this order: GAME -> PATCH -> DLC

    What You Need

    • Modded Ps4
    • USB device with at least 32GB of memeory
    • Game backup you want to install

    Prep Work

    • Make sure the USB flash drive is in exfat, you can check with Guiformat, or some other software that can check, and format the USB
    • Copy the game file to the root of the USB device
    • Plug the USB device into the Ps4 console

      • Instructions

      Section I - Running HEN

      1. Remove the hard drive from the console and unplug any USB devices before activating the exploit
      2. On the console, go to Settings in the "User's Guide / Useful Information", "User's Guide"
        • If an error occurs, restart the console
      3. Instead of the manual, you should get “PS4 Exploit Host by Al-Azif”
      4. At the end of the process, a notification will pop up
      5. Press OK
      6. Select “5.05”
      7. Select “HEN”
      8. When it says “You are all set” (or sometimes says “Done”) it means the exploit has been successful
        • If it does not exit the first time (Insufficient memory), press the (O) button to close the error message and try again
        • You can also clear the browser data and delete cookies, and then restart the console
      9. Press (PS) to exit to the startup screen
      10. Now you can connect all previously disconnected USB devices and hard drives

      Section II - Installing Games

      1. Open "Settings"
      2. Scroll down to "Debug Settings", and press (X) to open it
      3. Scroll down and press (X) to open the "Game" menu
      4. Scroll down and press (X) to open the "Package Installer"
      5. Select the game to install
        • If you selected several games, select "Install All"
      6. After finishing the installation, press "(PS)" to return to the home screen
      7. The game(s) should now be on the homescreen, now, enjoy playing GoW
